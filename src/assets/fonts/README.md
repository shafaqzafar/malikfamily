# Font Files Directory

This directory contains font files used in prescription PDFs and other components.

## Required Font Files:

### AlQalam Taj Nastaleeq (Urdu Font)
- **File**: `AlQalamTajNastaleeq.ttf`
- **Purpose**: Used for rendering Urdu instructions in prescription PDFs
- **Location**: Place the font file in this directory as `AlQalamTajNastaleeq.ttf`
- **Usage**: Automatically loaded when Urdu text is detected in prescription instructions

### Poppins Font Family
- **Regular**: `Poppins-Regular.ttf` (automatically downloaded from Google Fonts)
- **SemiBold**: `Poppins-SemiBold.ttf` (automatically downloaded from Google Fonts)
- **Purpose**: Primary font for prescription PDFs and UI components
- **Usage**: Used across all prescription templates for consistent typography

## Notes:
- The Urdu font file needs to be manually placed in this directory
- Poppins fonts are automatically fetched from Google Fonts CDN
- Make sure font files are properly licensed for use in medical applications
